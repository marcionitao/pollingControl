# pollingControl

pollingControl is a backend polling app that communicates in real time via websockts.

## Technologies

- TypeScript
- Node
- Fastify (backend framework)
- Prisma (ORM)
- Postgres (database)
- Redis (data caching)
- Zod (data validation)
- WebSocket (communication protocol)
